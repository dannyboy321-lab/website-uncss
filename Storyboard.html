Wireframe Layout

Below is a basic wireframe of how my website is structured. The website consists of six main pages: Index, Professional, Personality, Personal Development, Storyboard, and References.

Page Purpose and Features

  • Index Page: The homepage introduces the user to my website and provides links to the other sections.
  • Professional Page: Displays my resume and professional goals.
  • Personality Page: Highlights my personality traits and the results of my personality test.
  • Personal Development Page: Reflects on my personal growth and educational journey.
  • Storyboard Page: This page outlines the design of the website.
  • References Page: Contains all the sources and references used for the project.

Design Theme

The design theme uses clean, minimalistic elements with a consistent color palette to create a professional yet personal feel. The target audience for the website is potential employers and educators, showcasing my skills and development in a well-organized format.
